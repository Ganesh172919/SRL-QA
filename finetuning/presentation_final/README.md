# Final Conference Presentation Package

This folder stores the generated 40-slide final conference outputs.

## Build commands

- `python build_final_presentation_assets.py`
- `python build_presentation_docs.py`
- `python build_presentation_deck.py`

## Main outputs

- `MASTER_FINAL_PRESENTATION.md`
- `pdfs/MASTER_FINAL_PRESENTATION.pdf`
- `deck/QA_SRL_Final_Conference_40_Slides.pptx`
- `deck/QA_SRL_Final_Conference_40_Slides.pdf`

## Notes

- `manifest.json` is the single source for section order, outputs, slide allocation, and figure inventory.
- `sources.json` stores source links used for clickable research citations in survey/latest-work slides.
