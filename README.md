# NLP Project

This download contains the runnable project in:

- `srl_qa_project/`

To run the full pipeline, open a terminal in `srl_qa_project/` and run:

```bash
python -m pip install -r requirements.txt
python main.py --mode full
```

For a detailed walkthrough (data format, model, evaluation, PDF deliverables), see `srl_qa_project/docs/`.
